Edu 邮箱与 Cursor Pro 教育优惠安全使用指南

以下所有步骤均为确保 edu 邮箱安全有效，避免后续出现翻车问题。

一、邮箱登录与安全操作

  1. 登陆 IP 要求

    推荐使用住宅ip登录 比如台湾ip等

  2. 密码修改须知

学校限定修改密码必须使用学校的电脑，相当于无法改密码 ，原理上只有本店老板和购买人知道邮箱密码（脚本新鲜跑的，密码是学校根据生日自动生成的），cursor密码不受影响。

二、GitHub 绑定（推荐）

推荐使用 edu 邮箱注册 GitHub 账号，这样可以用 GitHub 登录 Cursor，进一步避免邮箱无法访问导致无法访问账户。
如果已经有github账号 也可以直接绑定到自己的账号上 （https://github.com/settings/emails）

  1. 注册 GitHub 账号时，使用 edu 邮箱进行注册。
  2. 登录 GitHub 后，绑定自己的常用邮箱作为备用邮箱。
  3. 在 GitHub 账号设置里启用「两步验证」，确保账号安全，即使密码泄露他人也无法登录。
  4. 完成上述安全设置后，等待数日再修改 edu 邮箱密码。

在确认完成上述所有安全设置后，按如下步骤升级 Cursor Pro 账号：

  1. 使用 edu 邮箱登录 Cursor 官网，进入账户页面。

  2. 点击「升级 Pro」按钮，如图所示：

    • 订阅页面会显示订阅金额为 $0：

  2. 完成订阅流程后绑定支付方式推荐使用信用卡+随便找一个美国地址，支付宝太大陆了。
